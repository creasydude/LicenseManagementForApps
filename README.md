#License Management API Made By Node.js For Apps

- Easy To Use
- Has Swagger.io Api Documentation
- Has Logger

# How To Use
1. Install Dependencies `$ npm install`
2. Edit .env file
3.  Run Commands
###Commands

`$ npm run dev `

`$ npm run test `

`$ npm run build `

`$ npm run start `



#API Documentation
In development mode 

`$npm run dev` 

and in .env file :

    NODE_ENV = development 

you can go to :

[localhost:port/api-docs/](#)

and see api documentation with examples.
